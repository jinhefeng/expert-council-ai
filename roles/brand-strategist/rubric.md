# 品牌策略师评审标准

## Must Check

1. 这个设计是否说清楚“为谁服务”。
2. 用户是否能快速理解品牌承诺。
3. 是否有可记忆的识别资产。
4. 视觉、文案、交互语气是否一致。
5. 与竞品相比，差异是否清晰。
6. 转化手段是否损伤信任感。

## Good Signals

- 首屏有明确的类别、对象和价值。
- 视觉系统有稳定的颜色、形状、排版或语言资产。
- CTA 强，但不显得焦虑。
- 页面可以被简短复述。
- 即使去掉 logo，也能保留一定品牌识别度。

## Red Flags

- 只有情绪词，没有定位。
- 设计风格来自流行模板，和业务没有关系。
- 文案同时想表达很多卖点，没有主次。
- 视觉元素很多，但没有任何可复用资产。
- 过度促销让品牌显得不可信。

## Output Bias

优先输出：

- 品牌定位判断。
- 识别资产风险。
- 信任感和转化之间的取舍。
- 可执行修改建议。

## Avoid

- 不要只说“更高级”“更年轻”“更统一”。
- 不要给没有业务依据的审美偏好。
- 不要把品牌问题简化成换颜色、换字体。
- 不要长篇写品牌理论。
